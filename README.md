# ✅ Sistema de Gestão de Tarefas com Flask

## 📌 Sobre o projeto

Aplicação web desenvolvida em Python com Flask para gerenciamento de tarefas, com autenticação de usuários e controle de acesso.

O sistema permite organizar atividades, acompanhar status e gerenciar informações de forma estruturada, simulando um cenário real de uso corporativo.

---

## 🎯 Objetivo

* Desenvolver uma aplicação web com autenticação
* Implementar operações CRUD completas
* Aplicar boas práticas de organização e segurança
* Simular um sistema real de gestão de tarefas

---

## 🛠️ Tecnologias utilizadas

* Python
* Flask
* SQLAlchemy
* SQLite
* Flask-Login
* Werkzeug (segurança)

---

## 🔐 Principais funcionalidades

### 👤 Usuários

* Cadastro, login e logout
* Proteção de rotas autenticadas
* Redefinição de senha (com token seguro)

### 📋 Tarefas

* Criação de tarefas
* Edição e atualização de status
* Exclusão de tarefas
* Listagem organizada

---

## 📊 Resultados e aprendizados

* Desenvolvimento de aplicação web completa
* Implementação de autenticação e controle de acesso
* Organização de código em camadas
* Aplicação de segurança em sistemas web

O sistema simula um ambiente real de gestão de tarefas com autenticação de usuários e controle de acesso, demonstrando aplicação prática de conceitos de backend.
